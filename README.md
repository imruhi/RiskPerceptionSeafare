# RiskPerceptionSeafare
Using NLP to look at risk perception in seafaring in historical evidence.

Texts are collected from [TOPOSText](https://topostext.org/), port/equivalent information is from deGraauw datasets, shipwrecks are from the [Oxford Geodatabase of Shipwrecks 1500BCE-1500CE.](https://oxrep.web.ox.ac.uk/shipwrecks-database). TODO: check if we can share the data?

Pipeline for creating the datasets can be seen in `data_gathering/utils/main.sh`, with related graphs in `data_gathering/utils/figs/`

(Python ver 3.11)
